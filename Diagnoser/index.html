Diagnoser

Oftalmologi

  • Penetrerende øjentraume
  • Akut glaukom
  • Arterirtis temporalis
  • Blow-out fraktur
  • Læsion af tåreveje
  • Baseætsninger
  • Keratit
  • Post-septal cellulitis

Hvad skal henvises kl. 3 om natten

  • Hul på øjet / penetrerende øjentraume
    • hård klap.
  • Akut glaukom
  • Alt andet kan ses i dagtid.
  • RING:
    • Keratit,
    • akut glaukom,
    • art. temp.,
    • central artie okklusion,
    • amotio

Neurologi

Der blev gennemgået meget specifikke ting

  • Diffus cerebral påvirkning pga. morfika
  • Coma
    • Diffus
    • Fokalt
  • Apopleksi
    • Malignt media-infarkt
    • Locked-in-syndrome
  • SDH
  • Kramper
    • Epilepsi
    • Status epilepticus
  • SAH
  • Herpes encefalit?

Tilstande nævnt på første slide

  • Bevidsthedssvækkelse — coma
  • Hemiparese
  • Bulbære symptomer med dobbeltsyn, dysartri, dysfagi, hemi- eller tetraparese, vertigo, balance- og koordinationssvigt
  • Neurologisk betinget respirationsinsufficiens (fx GBS, MG)
  • Svær hovedpine
  • Vertigo, ikke mono-symptomatisk
  • Kramper og status epilepticus
  • Konfusion og delir
  • Para- og tetraparese (fx medulært tværsnitssyndrom)
  • Neurogen urinretention eller cauda equina-symptomer

Respiratoriske tilstande

  • Pneumoni
  • KOL
  • Asthma
  • AMI
  • Lungeemboli
  • Lungeødem / -stase
  • Hjertesvigt
  • Pneumothorax
  • Atrieflagren
    • Kort snakket om under EKG.

3 Hyppige årsager til dyspnø

  • Pneumoni
  • Obstruktiv lungesygdom (KOL, astma)
  • Mb. cordis

    • Hallucinogener: Syre, svampe, LSD

    • Amatoxin
